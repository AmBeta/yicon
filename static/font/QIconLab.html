第一步：使用font-face声明字体 
  @font-face {font-family: 'QIconLab';
    src: url('QIconLab.eot'); /* IE9*/
    src: url('QIconLab.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('QIconLab.woff') format('woff'), /* chrome、firefox */
    url('QIconLab.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
    url('QIconLab.svg#iconfont') format('svg'); /* iOS 4.1- */
  }
第二步：定义使用iconfont的样式 
  .iconfont {
    font-family:"QIconLab" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;}
第三步：挑选相应图标并获取字体编码，应用于页面
